Use cases

Each use case has a realism-requirement profile over the 11 dimensions (critical / useful / not needed).

  • UC-A1 · Targeted data exfiltration

    Locate sensitive data and move it out of the network covertly, against monitoring and baselines.

  • UC-A2 · Ransomware deployment

    Deploy ransomware across hosts: propagation, encryption, and impact.

  • UC-A3 · Worm / self-propagation

    Self-propagating spread across a network from an initial foothold.

  • UC-A5 · Credential-based privilege escalation

    Escalate from a low-privilege account to higher privileges using credential and OS mechanics.

  • UC-A6 · Evasive red-team operation

    Conduct a stealthy, operationally-secure campaign that blends into realistic activity.

  • UC-D1 · Alert triage

    Triage security alerts, separating malicious activity from benign noise.

  • UC-D2 · Incident response

    Respond to and contain an active incident.

  • UC-D3 · Threat hunting

    Proactively hunt for threats across hosts and telemetry.

  • UC-S1 · Attacker-defender simulation

    Simulate interacting attacker and defender agents.

  • UC-S2 · Security architecture evaluation

    Compare configurations and policies for time-to-compromise and weakest paths.
